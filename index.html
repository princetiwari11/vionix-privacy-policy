VIONIX – Privacy Policy

Last Updated: November 2025

Vionix (“the App”) is developed by Avince Developers (“we”, “us”). This Privacy Policy explains how the App handles user data.

1. Data We Do NOT Collect

Vionix does NOT collect, store, upload, or share:

  • Personal information
  • Location history
  • Photos, videos, or audio files
  • Contacts or messages
  • Browsing history
  • User identifiers
  • Analytics or advertising data

2. Data Processed Locally (On Device Only)

The App accesses certain data strictly on the device and never sends it anywhere.

a. Storage Media (Photos, Videos, Audio)

Used only to calculate storage usage and categorize files. Never uploaded, stored, or shared.

b. Installed Apps List (QUERY_ALL_PACKAGES)

Used to analyze app storage usage and device performance. Allowed for device optimization apps. Not uploaded or shared.

c. Location (Optional)

Used only for the weather feature. Not stored or shared. Users can deny the permission.

3. VPN Disclosure

Vionix uses Android's local VPNService to enable security features.

The VPN does NOT:

  • Send traffic to external servers
  • Log network activity
  • Inspect or store traffic
  • Share user data

All processing happens locally on the device.

4. No Data Sharing

The App does not share any user data with any third party.

5. Data Retention

We retain no personal data. All temporary processing stays on the device.

6. Data Deletion

Since no data is stored, uninstalling the app removes all temporary cache.

7. Children’s Privacy

The app is not designed for children under 13. No child data is collected.
