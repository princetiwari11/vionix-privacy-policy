VIONIX — Privacy Policy

Last updated: 18 November 2025

1. Overview

VIONIX (“the App”) is a device optimization and privacy utility. Features include system performance tools, a storage analyzer (apps / media / system), DNS-based Guard Mode (local VPN), a weekly summary, and cybersecurity news.

3. Permissions & Data Usage

The app requests only the permissions necessary for core features. No data is collected or shared unless explicitly stated.

3.1 Internet

  • android.permission.INTERNET — Used for cybersecurity news, weather, and DNS resolution in Guard Mode.

3.2 Media

  • READ_MEDIA_IMAGES / VIDEO / AUDIO — Used to calculate media storage size. Only metadata (size, count) is used.
  • READ_EXTERNAL_STORAGE — For older devices only. Used for the same purpose.

3.3 Location

  • ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION — Used only for the optional Weather feature. The app does not store location.

3.4 Installed Apps & Usage

  • QUERY_ALL_PACKAGES — Required for the Storage Analyzer to show system/user installed apps. Processed locally only.
  • PACKAGE_USAGE_STATS — Used to show app usage and performance suggestions. Data never leaves the device.

3.5 Foreground Services & Notifications

  • FOREGROUND_SERVICE — Required for persistent background execution.
  • FOREGROUND_SERVICE_VPN — Required for Guard Mode (local VPN service).
  • POST_NOTIFICATIONS — Needed to show active Guard Mode notification.

4. Guard VPN (DNS-only)

The Guard Mode uses Android’s VpnService to provide secure DNS routing.

  • Does not monitor, track, or inspect traffic.
  • Does not block content unless user adds domains manually.
  • Does not log DNS queries.
  • Uses Cloudflare or system DNS resolvers — no developer server involved.
  • Everything is processed locally on device.

5. Data We Do NOT Collect

VIONIX does not collect:

  • Personal information
  • App usage history
  • Location history
  • Contacts, messages, photos
  • Identifiers or analytics data
  • VPN traffic or browsing data
  • Installed apps list (never uploaded)

6. Third-Party Sharing

The app does not share any data with third parties. External services are contacted only for:

  • Cybersecurity news (public RSS feed)
  • Weather (if enabled)

7. Encryption

All outbound traffic (weather/news/DNS) uses encrypted HTTPS or DoH where available.

8. Data Retention & Deletion

  • App data stays on-device and is deleted when the user uninstalls the app.
  • No personal data is stored on servers.
  • Users may request deletion of any voluntarily provided data via email.

9. Children’s Privacy

The app is not intended for children under 13 and does not collect data from children.

10. Changes

We may update this policy for legal or functional reasons. The date above will be updated accordingly.

Reviewer Note: All permissions directly match the app manifest. Media, location, and usage stats are optional and used only for on-device features. Guard Mode is DNS-only, no traffic inspection or logging.
